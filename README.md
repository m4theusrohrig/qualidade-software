## 🧩 Metodologia

**Aprendizagem Baseada em Problemas (PBL)**, com foco no desenvolvimento de
competências em qualidade de software aplicadas ao projeto **Local Eats**.

O problema norteador simula uma situação real: uma startup lançou rapidamente
uma plataforma digital e precisa de uma equipe de qualidade para diagnosticar
e corrigir seus problemas com base na norma **ISO/IEC 25000**.
